### User accept 

```SQL
WITH 
    sentReq AS ( 
    select *
    from fb_friend_requests
    where action = 'sent'),
   
    accpReq AS ( 
    select *
    from fb_friend_requests
    where action = 'accepted'
   )
 
SELECT s.date, a.date
FROM sentReq s
left join accpReq a on s.user_id_sender = a.user_id_sender
```
